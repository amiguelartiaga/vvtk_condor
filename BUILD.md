## Building installers

Build each package from its own folder script.

### voz

```bash
bash src/voz/build_installer.sh
```

This produces the single-file `condor_voz` installer from `src/voz/`.

### hermes2

```bash
bash src/hermes2/build_installer.sh
```

This produces the single-file `condor_hermes2` installer from `src/hermes2/`.
